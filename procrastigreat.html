01
Background

The Challenge

During a UI course, I was given the opportunity to identify a user demographic that would benefit from a mobile productivity app and create interface visuals to support their needs. I chose to support neurodivergent Gen-Z students.

Neurodivergent individuals have conditions such as autism spectrum disorder, ADHD, dyslexia, and more. As awareness of these conditions continues to grow, more and more Gen-Z students are being diagnosed every day.

Neurodivergent Gen-Z students value planning but find that current apps overwhelm and stress them out. How might we help them plan with a mobile app that feels calming instead?

53%

of Gen-Z say they identify as neurodiverse — Pew Research Center, 2023

65%

02
Discovery

Research & Insights

Secondary research and competitive analysis uncovered key insights about neurodiversity, Gen-Z students, and their pain points related to digital planner features. Two areas guided every design decision: visual design and feature needs.

Neurodivergent Visual Design Checklist

  • Sans-serif fonts
  • Pastel or muted color palettes
  • Clear visual hierarchy
  • Soothing, encouraging content & visuals
  • Accessible contrast ratio
  • Readable text at all sizes

Feature Needs

  • Task chunking & prioritization
  • Processing and planning tool variety
  • Habit growth visualization over time
  • Planned breaks reduce stress
  • Engagement without overwhelm
  • Reframe progress, not failure

A competitive analysis of popular productivity apps — Streaks, Apple Freeform, Notion, Toggl, and Apple Notes — revealed the feature set that would make ProcrastiGreat both familiar and distinctly better suited to this audience.

  • Streaks

    Circular progress indicators with editable icons and linked data graphs for habit progress over time.

  • Notion

    Smart table filters with Kanban board and list toggle options for to-do items.

  • Toggl

    Pomodoro timer with circular progress indicator for task anchoring to time charts.

  • Apple Notes

    Minimal note taking with hashtag-based categorization features.

03
Process

Wireframes & Iteration

Using research insights, planner app features were prioritized, sketched, and translated into low-fidelity wireframes and user-centered structures. The app covers nine core screens: Login, Home, Kanban To-Do, Notes, Whiteboard, Pomodoro Timer, Habits Tracker, Profile, and Stats.

04
High Fidelity

Visual Design

With a clear app structure in place, I moved into type, color, and visual experience. A moodboard, style guide, and custom icon set shaped the soothing aesthetic before a single high-fidelity screen was built.

05
Outcomes

Results & Impact

With strong user research, competitive analysis, and a clear structure, the visual design came together efficiently. The final ProcrastiGreat app is a complete eight-screen iOS UI ready for user testing and developer handoff — all files annotated for a smooth transition.

  • Login / Sign Up Screen
  • Home / Dashboard Screen
  • To-Do List with Kanban Board View
  • Notes with Hashtag Organization
  • Whiteboard for Brainstorming
  • Pomodoro Timer with Task Anchoring
  • Habit Tracker with Progress Visualization and hand-made icons
  • Profile & Settings Screen

  • First Icon Set

    Designed and iterated a complete custom icon set from scratch — a new skill built during this project.

  • Accessibility Challenges

    Learned firsthand how pastel palettes create contrast challenges — and how to solve them without losing the calming aesthetic.

  • Ready for Handoff

    All screens annotated and structured for a clean developer handoff — needs only user testing before build.

This project sharpened my visual design eye considerably — from building a custom icon set to navigating the tension between soothing aesthetics and accessibility contrast requirements. Every iteration made the app more genuinely useful for the students it was designed to serve.
