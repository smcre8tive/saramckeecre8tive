ProcrastiGreat Gen-Z Mobile Planning App

Neurodiverse Gen-Z Students Can Procrastinate Powerfully with these calming interface designs.

Role: UI Designer | Duration: 2 weeks | Tools: Figma

The ProcrastiGreat productivity app.

Challenge & Problem

uring a UI course, I was given the opportunity to identify a user demographic that would benefit in 2024 from a mobile productivity app and create interface visuals to support their needs. I chose to support neurodivergent Gen-Z students.

Neurodivergent individuals have conditions such as autism spectrum disorder, attention deficit or hyperactivity disorder (ADHD), dyslexia, and more. As awareness of these conditions continues to grow, more and more Gen-Z students are diagnosed every-day.

Neurodivergent Gen-Z students value planning but find that current planning mobile apps overwhelm and stress them out. How might we help Neurodivergent, Gen-Z students plan with a mobile app that does not overwhelm or stress them out?

Research & Discovery

Secondary research and competitive analysis helped uncover insights about neurodiversity, Gen-Z stuents, and their pain-points related to digital planner features.

In a recent survey, 53% of Gen-Z say they identify as neurodiverse. Source: Pew Research Center, 2023
65% of Gen-Z students report feeling overwhelmed by traditional planning apps due to their complexity and cluttered interfaces. Statista
Procrastigreat, for the great procrastinator, leans on soothing self compassion driven visuals.

Digging into neurodivergent & Gen-Z design needs identified essential features and UI patterns that would support this audience. Key findings included:

Neurodivergent Visual Design Check List

  • San-Serif fonts
  • Pastel or muted color palettes
  • Clear visual hiearchy
  • Soothing, encouraging content & visuals
  • Accessible contrast ratio and text readabiity

Neurodivergent Visual Design Check List

  • Task chunking & prioritization breaks down overwhelm
  • Processing and planning tool variety increases engagement
  • Habit growth over time helps reframe without triggering anxiety
  • Planned breaks reduce stress and increase focus ability during tasks

To-do lists with optional automated reminders are essential tools for neurodivere Gen-Z students.

A comparison of popular productivity mobile apps revealed insights that would help to create competitive features.

Competitive Features From Popular Planning Apps

  • 1. Streaks- Circular progress indicators with editable icons to identify each habit and linked data graphs to show habit progress over time
  • 2. Apple Freeform- Shareable whiteboard with brainstorming tools
  • 3. Notion- Smart table filters with Kanban board & list information toggle options for to-do items
  • 4. Toggl- Pomodoro timer with reduction oriented circular progress indicator screen for task anchoring to time charts
  • 5. Apple Notes- Minimal note taking with hashtag based categorization features

These features from the popular planning apps mentined above, will help support Gen-Z students and help the app have competitive features.

Early mid-fidelity wireframes helped refine user flows and screen structures.

Results & Impact

With strong user research, competitive analysis, and a clear structure, the visual design came together quickly. The final ProcrastiGreat app includes:

  • Login/Sign Up Screen
  • Home/Dashboard Screen
  • To-Do List with Kanban Board View
  • Notes with Hashtag Organization
  • Whiteboard for Brainstorming
  • Pomodoro Timer with Task Anchoring
  • Habit Tracker with Progress Visualization and hand-made icons
  • Profile & Settings Screen

The original High-fidelity mobile app screens for ProcrastiGreat needed more contrast for accessiblity and a stronger title font.
The Final edit to the mobile app screens for Procrastigreat shows higher color and type font contrast.
Selected user feedback and testimonials from post-launch surveys

With strong research, ideation and design, this user interface design for the Procrastigreat mobile app planner just needs user testing and a developer handoff. All files and features are prepared with annotations for an easy hand-off.

I learned a lot about user interface design during this project. From creating my first icon set, to struggles with accessibility for pastel colors, I learned how to improve my visual design eye for muself and Gen-Z neurodiverse students.
